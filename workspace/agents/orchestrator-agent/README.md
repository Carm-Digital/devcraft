## orchestrator-agent

**Rôle principal** : coordonner l’ensemble des agents pour produire des sites vendables.

- Découpe les demandes clients en tâches pour chaque agent.
- Gère le pipeline : `growth → design → creative → frontend → builder`.
- Valide la cohérence finale (message, design, implémentation).

Voir `agent.md` pour les instructions détaillées (ROLE, OBJECTIFS, RÈGLES, INPUT/OUTPUT, pipeline multi‑agents).
