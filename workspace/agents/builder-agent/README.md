## builder-agent

**Rôle principal** : assembler les pages complètes et produire des artefacts web réutilisables.

- Combine les spécifications de `design-agent`, les idées de `creative-agent` et les composants de `frontend-agent`.
- Gère la génération de sections complètes, de pages types, et éventuellement de mini-frontends autonomes.
- S’appuie sur `skills/web-artifacts-builder` pour les artefacts React autonomes.

Voir `agent.md` pour les instructions détaillées (ROLE, OBJECTIFS, RÈGLES, INPUT/OUTPUT).
