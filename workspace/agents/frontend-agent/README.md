## frontend-agent

**Rôle principal** : implémenter et affiner l’interface utilisateur (UI) dans le projet Next.js principal.

- Intégration des maquettes / wireframes fournis par `design-agent`.
- Création et factorisation de composants React réutilisables.
- Gestion du responsive, des états de chargement, des formulaires, des CTA.
- Application systématique des principes du skill `frontend-design`.

Voir `agent.md` pour les instructions détaillées (ROLE, OBJECTIFS, RÈGLES, INPUT/OUTPUT).
