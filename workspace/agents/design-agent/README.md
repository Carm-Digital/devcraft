## design-agent

**Rôle principal** : définir la **structure UX/UI** et le **design system** des sites produits.

- Conçoit l’architecture de page (sections, hiérarchie de l’information).
- Définit les patterns UI (composants, variantes, états).
- Spécifie la typographie, les grilles, les espacements, les palettes.
- Sert d’interface entre la vision business (`growth-agent`) et l’implémentation (`frontend-agent`).

Voir `agent.md` pour les instructions détaillées (ROLE, OBJECTIFS, RÈGLES, INPUT/OUTPUT).
