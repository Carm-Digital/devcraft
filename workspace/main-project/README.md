## main-project

Ce dossier logique représente le **projet Next.js principal**.

> Le code vit déjà à la racine du repo (`src/`, `next.config.ts`, etc.).  
> Pour éviter de tout déplacer et risquer de casser l’app, on garde le code en place et on documente simplement ce mapping.

- Racine code : `./` (racine du repo)
- Type de produit : **Agence de création de sites web** (DevCraft).
- Stack : Next.js, React, Stripe, Tailwind‑like utility classes.
- Pages clés :
  - `src/app/page.tsx` : landing principale “Agence web / création de site en 7 jours”.
  - `src/app/services`, `src/app/methode`, `src/app/realisations`, `src/app/a-propos`, etc.

Ce dossier sert de **référence** pour les agents : quand un agent parle de `main-project`, il fait référence au code de la racine actuelle.
